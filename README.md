你是一名资深 Python 视频工程师，请生成一个【完整、可运行】的 Python 脚本，用于根据 input.json 自动合成视频。

【技术要求】
1. 使用 uv 管理 Python 环境（uv 已安装）
2. 代码需结构清晰、可直接执行

【input.json 数据结构】
{
  "list": [
    { "cap": "字幕内容 1" },
    { "cap": "字幕内容 2" }
  ],
  "audio_list": [
    "音频文件 URL 1",
    "音频文件 URL 2"
  ],
  "duration_list": [
    3200,
    2800
  ],
  "image_list": [
    "图片文件 URL 1",
    "图片文件 URL 2"
  ]
}

字段说明：
- list.cap：字幕文本
- audio_list：音频 URL 列表
- duration_list：音频时长列表（单位：毫秒）
- image_list：图片 URL 列表
- 四个列表长度一一对应

【视频生成规则】
1. 视频总时长 = 所有音频时长之和
2. 每个音频片段：
   - 播放对应音频
   - 显示对应字幕
   - 使用对应图片作为背景
3. 图片需生成「随机抽帧动画效果」，例如：
   - 轻微缩放（zoom in / out）
   - 轻微位移（pan / shake）
   - 在该音频时长内形成平滑动态效果
4. 字幕展示规则：
   - 仅在对应音频播放期间显示
   - 底部显示
   - 白色字体，带描边或半透明背景
   - 自动换行

【脚本功能要求】
1. 自动下载所有音频和图片资源
2. 按 input.json 顺序合成多个视频片段
3. 拼接为一个完整视频文件（output.mp4）
4. 提供 uv 环境创建与依赖安装命令示例
5. 提供运行脚本的命令示例

【输出要求】
- 只输出一个完整的 Python 脚本
- 脚本可直接运行，无需手动修改
